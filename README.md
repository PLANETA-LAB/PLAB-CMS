## 🌌 О CMS

PLAB CMS — это легковесная, безопасная и современная система управления контентом, созданная специально для музыкальных дистрибьюторов, лейблов и артистов.

### Особенности

- 🚀 **Быстрая установка** — установщик как у 1С-Битрикс
- 🔒 **Безопасность** — защита от копирования и изменений
- 🎨 **Космический дизайн** — уникальный стиль PLANETA LAB
- 📀 **Управление релизами** — полный цикл от загрузки до модерации
- 🎵 **Поддержка WAV** — загрузка и прослушивание
- 👥 **Личный кабинет артиста** — управление своими релизами
- ⚡ **Статусы релизов** — Черновик, Модерация, Отгрузка, Принят, Отклонен
- 🤖 **AI-треки** — поддержка загрузки AI-контента (опционально)
- 🎤 **Караоке синхронизация** — TTML редактор

## 📋 Минимальные требования

| Компонент | Минимальная версия | Рекомендуемая |
|-----------|-------------------|---------------|
| **PHP** | 7.4 | 8.1+ |
| **MySQL** | 5.7 | 8.0+ |
| **Память** | 128MB | 256MB |
| **Загрузка файлов** | 64MB | 128MB |
| **Apache** | 2.4 | 2.4+ (mod_rewrite) |
| **OpenSSL** | 1.1.0 | 1.1.1+ |

### Требуемые расширения PHP

- `pdo_mysql` — для работы с БД
- `gd` — для обработки изображений
- `json` — для работы с данными
- `mbstring` — для многобайтовых строк
- `openssl` — для безопасности
