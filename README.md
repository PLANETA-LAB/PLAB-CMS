## 🌌 О проекте

PLAB CMS — легковесная CMS для музыкальных дистрибьюторов, лейблов и артистов. Создана для управления релизами, модерации и дистрибуции музыки на цифровые площадки.

### Особенности

- 🚀 **Быстрая установка** — установщик как у 1С-Битрикс
- 🔒 **Безопасность** — защита от копирования и изменений
- 🎨 **Космический дизайн** — уникальный стиль
- 📀 **Управление релизами** — полный цикл: черновик → модерация → отгрузка → принят
- 🎵 **Поддержка WAV** — загрузка и прослушивание
- 👥 **Личный кабинет артиста** — управление своими релизами
- 🤖 **AI-треки** — поддержка AI-контента (опционально)
- 🎤 **Караоке синхронизация** — TTML редактор

## 📥 Установка

### Минимальные требования

| Компонент | Версия |
|-----------|--------|
| PHP | 7.4+ |
| MySQL | 5.7+ |
| Память | 128MB |
| Загрузка | 64MB |

### Шаги установки

1. Скачайте архив с [GitHub Releases](https://github.com/PLANETA-LAB/PLAB-CMS/releases)
2. Загрузите файлы на хостинг
3. Откройте `https://ваш-сайт.ru/install.php`
4. Следуйте инструкциям установщика
